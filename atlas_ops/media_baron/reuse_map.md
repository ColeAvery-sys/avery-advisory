# Media Baron Reuse Map

One approved script should become a reusable content asset, not a one-time post.

## Reuse Flow

1. Long YouTube video: full idea, narrative, proof, CTA, and authority.
2. Shorts: five to ten high-hook moments from the script.
3. Instagram reel: short version with clear visual direction.
4. Facebook post: direct explanation for local/business audience.
5. Blog post: search-friendly version with internal links.
6. Newsletter: personal founder update or useful breakdown.
7. Sales page copy: offer-focused version if the topic supports a service.
8. Voiceover: ElevenLabs generation only after script approval and credit check.
9. Quote cards: clean lines for social assets.
10. Reddit discussion post: community-specific, non-spam version.
11. Product/service promo: only if the idea naturally supports an approved offer.

## Guardrail

Reuse is not spam. Every platform version should respect audience context, platform rules, asset rights, and approval status.
