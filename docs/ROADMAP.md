# Roadmap

## Current North Star
Build `ATLAS HQ` into the operating system for `Avery Industries LLC`.

## Company Network Track
Build the Avery Industries LLC mother page and shared brand network so every division can launch from one platform.

## Priority Order
1. ATLAS HQ executive dashboard
2. Approval gate system
3. Revenue automation
4. Lead generation and CRM
5. Content operations pipeline
6. Grant operations workflow
7. Idea vault and memory indexing
8. Division dashboards
9. Reporting and audit logs
10. External integrations with approval controls
11. Real estate wholesaling launch lane
12. Career OS master profile, LinkedIn sync, and job-application outputs
13. ATLAS Recruiter daily job ranking, draft applications, and follow-up tracking
14. Avery Industries LLC mother page and company network routing

## Working Principles
- Build the core control plane before feature sprawl.
- Keep every external action gated by user approval.
- Make the default view a real working surface, not a marketing page.
- Prefer clear data tables, concise summaries, and dense task surfaces.
- Start real estate with wholesaling before rentals or development.

## Status Cadence
After each phase:
- update the implementation notes
- update the QA checklist status
- update the roadmap status
- log what changed and what still needs work

## Release Tiers
- Draft: exists locally only
- Internal: validated and usable by Avery Industries LLC
- Release-ready: checked, logged, and approved for use
- Public-facing: ready for client or external use with final approval
