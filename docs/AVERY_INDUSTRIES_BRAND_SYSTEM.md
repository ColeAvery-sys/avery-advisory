## Rebrand Rule

Do not rebrand every old HTML page manually without a shared CSS pass.

Correct approach:

1. Create or update a shared brand CSS file.
2. Make every old dashboard page load that CSS.
3. Standardize header, navigation, colors, button styles, cards, and ATLAS dock/chatbox.
4. Validate all links.
5. Then update page-specific layout only where needed.

## Smart Glasses / Disability-Aid Brand Rule

Any facial, hand-signal, camera, GPS, microphone, or disability-aid feature must be framed as:

- Opt-in.
- Privacy-first.
- Approval-gated.
- Local-first where possible.
- Assistive support, not medical diagnosis.
- Evidence-backed before expansion.
