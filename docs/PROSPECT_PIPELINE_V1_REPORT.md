# Prospect Pipeline V1 Report

Date: 2026-06-18

## Objective

Create a prospect tracking system for Avery Advisory and Creator Logistics so outreach can move from random messages to an organized target acquisition pipeline.

## Files Created

- `docs/sales/prospect-pipeline-template.csv`
- `docs/sales/prospect-research-guide.md`
- `docs/sales/prospect-scoring-system.md`

## Usage Instructions

### Prospect Pipeline Template

Use the CSV to track leads, contact method, audit status, discovery calls, proposals, and client wins.

### Prospect Research Guide

Use this to identify the right creators and local businesses before outreach.

### Prospect Scoring System

Use the score to sort prospects by likely fit and responsiveness.

## Suggested Workflow

1. Research prospects
2. Score prospects
3. Add them to the CSV
4. Send outreach
5. Log replies
6. Send audits
7. Book discovery calls
8. Send proposals
9. Mark wins and losses
