# AveryTech Landing Batch 1

## Scope

Built a polished React + Vite + Tailwind landing site for AveryTech, the technology division of Avery Industries LLC.

## Validation Checklist

- `npm install` completed successfully
- `npm run dev` starts the Vite app on port 3000
- `npm run build` generates a production build
- Navbar, hero, products, mission, labs, investors, and footer all render
- Mobile layout collapses cleanly without broken navigation
- All requested product copy and ATLAS Mark I specs are present
- No lorem ipsum or placeholder copy is used
- No broken imports are left in the React tree
- Tailwind v4 is wired through `@tailwindcss/vite` and `@import "tailwindcss";`

## Status

Batch 1 is implemented and ready for local verification.
