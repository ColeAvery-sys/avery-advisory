# ADHD Start Here

Date created: 2026-06-02

Purpose: reduce visual overload and make the Avery Industries / ATLAS workspace usable when energy, attention, or working memory is low.

## If You Are Overwhelmed

Open only these files:

1. `AVERY_INDUSTRIES_AI_OS_CHARTER.md`
2. `AVERY_INDUSTRIES_SPRINT_ALPHA.md`
3. `AVERY_INDUSTRIES_MASTER_TASK_BOARD.md`
4. `docs/SYSTEMS_INDEX.md`
5. `docs/FILE_DATE_INDEX.md`

Do not open batch files unless you are actively building that batch.

## Today's Default Priority

1. Creator Logistics revenue.
2. Executive Dashboard / Easy Mode.
3. First editor or contractor pipeline.
4. ATLAS Core and memory.
5. ATLAS Assist and grant readiness.

Everything else is parking lot unless Cole explicitly promotes it.

## Workspace Zones

- Root `.ts` files: backend system modules.
- Root `BATCH_*.md` files: roadmap archive and build packets.
- `atlas_ops/`: operating documents, prompts, logs, templates, and local business files.
- `docs/`: repo maps, indexes, environment notes, and ADHD navigation.
- `Executive Reports/`: CEO-facing report outputs.
- `scripts/`: local maintenance and update scripts.

## Daily Operating Loop

1. Check `Executive Reports`.
2. Check the Master Task Board.
3. Pick one revenue move.
4. Clear urgent approvals.
5. Move one active task forward.
6. Stop before opening a new roadmap batch.

## Decision Rule

If a task does not help with revenue, executive control, client delivery, editor/contractor capacity, ATLAS Core, company memory, or grant readiness, it is probably not today's task.

## Low Energy Mode

Do one of these:

- Add one lead.
- Update one approval.
- Write one executive report row.
- Move one task from `Working` to `Review`.
- Archive one distracting idea into the Idea Vault.

One clean action counts.
